Разметь текст по схеме BIO, используя только указанные классы сущностей:
Activity, Application, Dataset, Environment, InfoResource, Lang, Method, Method_ML, Metric, Model, Object, Organization, Person, Science, Subject, Task.

Каждое слово должно быть помечено одной из меток:
- "B-<сущность>" — начало сущности,
- "I-<сущность>" — продолжение сущности,
- "O" — слово вне сущностей.

Важно:
1. Используй только указанные классы. Не добавляй новых классов.
2. Сохраняй слова текста в их исходной форме, не изменяй их грамматические формы.
3. Не переводи слова исходного текста на другой язык.

Примеры ошибок, которых следует избегать:
- "Недавно рассказывал о том, как делать иерархически датасет из Википедии." - здесь слово "датасет" не является сущностью класса "Dataset"
- "Для наглядности оставляю здесь иллюстрацию с последовательностью работы со Снокрелем для задачи information extraction из оригинальной статьи." - здесь слово "задачи" не входит в сущность класса "Task", сущностью является "information extraction"
- "Тогда модель определяется так, чтобы обучить эту модель без доступа к истинным меткам, это нужно обучаться с помощью логарифмического негативного маргинализированного правдоподобия, зная матрицу." - здесь слово "модель" не является сущностью класса "Model".
- "Загрузим заранее обученную модель fastText, чей выбор объясняется наличием огромного количества опечаток в текстах." - здесь слово "модель" не входит в сущность класса "Model", сущностью является "fastText".

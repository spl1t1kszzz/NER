Ты решаешь задачу разрешения кореференций (coreference resolution), используя подход Tree-of-Thoughts (дерево рассуждений). Это значит, что для каждого упоминания сущности ты должен рассмотреть несколько возможных вариантов и выбрать наиболее подходящий.

Используй следующую стратегию:

1. **Найди все потенциальные упоминания сущностей** в тексте.
2. Для каждого найденного упоминания **рассмотри несколько возможных вариантов** того, к какой сущности оно может относиться.
3. Используя логику и контекст, **выбери лучший вариант** для каждого упоминания.
4. В конце объедини упоминания в отдельные кластеры сущностей.
5. Верни результат ТОЛЬКО в виде массива из массивов, где каждый внутренний массив представляет собой кластер упоминаний одной сущности.

Пример рассуждения (Tree-of-Thoughts):

Текст:
«Иван переехал в Новосибирск. Он нашел там работу. Город оказался очень уютным, и Иван быстро привык к нему.»

Рассуждение:
- "Иван": это отдельная сущность (человек).
- "Он": варианты: ["Иван", другая сущность]. Контекст говорит, что это именно Иван.
- "Новосибирск": отдельная сущность (место).
- "там": варианты: ["Новосибирск", другое место]. По контексту это явно Новосибирск.
- "Город": варианты: ["Новосибирск", другой город]. Это точно Новосибирск.
- "нему": варианты: ["Новосибирск", другой объект]. Контекст указывает на Новосибирск.

Итоговый ответ:
[
  ["Иван", "Он"],
  ["Новосибирск", "там", "Город", "нему"]
]

Теперь выполни аналогичный анализ (с деревом рассуждений) для следующего текста:

"<ТВОЙ ТЕКСТ ЗДЕСЬ>"
